# cifar10-image-classifier
CNN image classifier on CIFAR-10 using PyTorch 
10 categories of images with ~75% test accuracy.

**Tech:** Python, PyTorch, torchvision, Google Colab
**Dataset:** CIFAR-10 (60,000 images, 10 classes)
**Architecture:** 3 Conv layers + BatchNorm + Dropout + 2 FC layers
**Result:** 75.4% test accuracy

### How to run
1. Open the .ipynb file in Google Colab
2. Runtime → Change runtime type → T4 GPU
3. Run all cells (Ctrl+F9)
